Final Project

"SHAPES!" is a program that allows users to draw polygons, ellipses, and lines. To create a shape, clickone of the buttons in the bottom left-hand corner of the screen. To create a vertex, users click on thedrawing area, a white 300x300 white canvas in the center of the screen. Users right-click to finalizetheir shape, and if a user regrets their decision, they can press "z" to undo it. Users can see thecomplete list of features and keybindings by clicking the "i" button in the bottom right-hand corner.The user has five layers to determine the order in which shapes appear, with the lowest layer being thefifth and the highest layer being the first. Users can select a layer by clicking on its correspondingnumber in the rightmost menu. To hide a layer, users can right-click on its corresponding icon. The user also has a color palette with 11 options for colors.Right-clicking on a swatch allows the user to choose a custom color.
